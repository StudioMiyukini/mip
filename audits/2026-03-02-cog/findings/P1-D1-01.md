## 10. Recommandations prioritaires

1. **miou-llm-bridge** : Refactorer les 30 unwrap() sur locks. Pattern suggere :
   ```rust
   let agents = self.agents.read()
       .map_err(|e| BridgeError::LockPoisoned(format!("{e}")))?;
   ```

2. **jaykonta/service.rs** : Remplacer le `panic!()` par un `Result` + log error.

3. **apps/central** : Corriger les 6 unwrap() user-facing (audio, jaymanga, jayfestival).

4. **Tests** : Prioriser les tests pour `jaykonta`, `lord_of_the_castle`, `kindmother-client/service`, `miyauth`.

5. **errors.rs** : Creer un type d'erreur pour les 13 Cores qui n'en ont pas.

6. **dead_code** : Audit des 38 `#[allow(dead_code)]` dans `apps/central` -- nettoyer ou justifier.
