## Actions prioritaires

1. **[CRITICAL] jayshop `run_blocking` recursion infinie** (#13) — Corriger `Err(_) => get_runtime().block_on(f)` dans `data/kindmother_client_db.rs:66`. Stack overflow en production si appele hors runtime tokio.

2. **[MAJOR] jayshop TODO stubs** (#15) — Implementer le mapping des 5 fonctions de lecture qui jettent les resultats SQL. Fonctionnalites actuellement cassees (retournent toujours des listes vides).

3. **[MAJOR] jayshop + jayfaim : zero tests** (#8, #16) — Ajouter des tests unitaires. Priorite jayshop (crate plus complexe, 15 types, schema SQL).

4. **[MAJOR] jayrdv `unwrap()` en production** (#1) — Correction simple : remplacer le `unwrap()` par un fallback non-faillible dans `slot_hold`.

5. **[MAJOR] jayshop clippy pedantic desactive** (#17) — Reactiver pedantic et corriger les warnings.

6. **[MAJOR] jayfaim annotations MSCM** (#9) — Ajouter les headers MSCM sur les 5 fichiers.
